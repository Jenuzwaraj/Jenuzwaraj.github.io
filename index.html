👨‍💻 About Me

I'm Jenuzwaraj, an Embedded & IoT Engineer with a deep passion for building real-time systems, optimizing performance, and solving complex hardware-software integration challenges.

With hands-on experience in microcontroller programming, embedded Linux (Yocto, Buildroot), and industrial IoT systems, I enjoy turning raw boards into production-ready smart devices. My journey involves working on everything from bootloaders and kernel customization to writing device drivers and deploying OTA updates.

This blog is my public lab notebook — sharing real projects, personal experiments, tutorials, and open source ideas. Whether you're a beginner trying to set up your first Linux build or a fellow engineer debugging weird SPI behavior at midnight — you’ll probably find something here for you.

📌 What to Expect from This Blog

  • Step-by-step guides on embedded Linux development
  • IoT project builds with schematics and source code
  • Boot-time optimization and real-time debugging tips
  • Open-source tools and custom driver tutorials
  • Behind-the-scenes notes from actual engineering work

If you're someone who loves building from the bare metal up — welcome aboard. Let's tinker, break, and learn together.
